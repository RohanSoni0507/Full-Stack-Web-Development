Login Form Project

Developer: Rohan Soni

Description:
A simple responsive login form centered on the screen. Built using only HTML and CSS.
This project demonstrates:
- Proper use of the box model (box-sizing, padding, border)
- Vertical and horizontal centering with Flexbox
- Consistent styling and spacing
- Beginner-friendly clean code

Tested on: Chrome, Firefox, Edge

Instructions:
Open index.html in any browser.
